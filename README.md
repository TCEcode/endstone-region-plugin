# RegionPlugin для Endstone — минимальный README

Коротко: плагин управления регионами для Endstone (Minecraft Bedrock).

Требования:
- Endstone 0.10.18+
- Python 3.12+

Установка:
1. Скопировать папку плагина в plugins/RegionPlugin на сервере Endstone.
2. Перезагрузить сервер.

Быстрые команды:
- `/rg pos1` — установить первую точку.
- `/rg pos2` — установить вторую точку.
- `/rg create <имя>` — создать регион.
- `/rg delete <имя>` — удалить регион.
- `/rg trust <регион> <игрок>` — дать доступ.

Конфигурация и база:
- Файл конфигурации: plugins/RegionPlugin/config.yml
- База данных: plugins/RegionPlugin/regions.db
